About CampusPilot

CampusPilot is a role-based campus management platform designed to streamline communication, data handling, and user interaction within an educational institution.

Main Roles & Permissions

  • Admin: Full access to the system, including managing all users, assigning roles, and configuring settings.
  • Editor: Can create, update and add details of new registered students, but cannot delete students.
  • Teacher: Can create new students and add details of new registered students, typically for academic purposes.
  • Student: Can log in to view and update their profile, and access personalized academic data.

Key Features

  • Role-based dashboards with unique menu items for each user type
  • Responsive design for mobile, tablet, and desktop
  • Secure authentication and role management
  • Automatic redirection to the home page on first visit

Dashboards & Navigation

Each role has its own dashboard and menu bar, showing only the features relevant to their permissions. This ensures a simple, focused, and secure experience.

How It Works

  1. New users register via the platform.
  2. Admin reviews the account and assigns the appropriate role (default is Student).
  3. If student details exist, no one can change its role.
  4. Upon login, the user is redirected to their dashboard acc to role.
